# Los Cinco de la Tierra - Imágenes de Personajes

Esta carpeta contiene las imágenes de "Los Cinco de la Tierra" para los cuentos de "Aprender Inglés".

## 📸 Imágenes Requeridas

Todas las imágenes deben estar en **estilo vintage storybook** (acuarela, colores cálidos, ilustración clásica de libro infantil).

1. `sara.png` - Sara (Barcelona, España 🇪🇸) - Adolescente, le gustan los videojuegos

3. `eva.png` - Eva (China 🇨🇳) - Adolescente, divertida, tiene un gato
4. `robert.png` - Robert (Estados Unidos 🇺🇸) - Adolescente, deportista
5. `gabriel.png` - Gabriel (Australia 🇦🇺) - Adolescente, tímido, ama ciencia ficción

### Personaje Secundario (Prioridad 2):
6. `oliver.png` - Oliver - Periodista adulto, amigo del equipo

## 🎨 Especificaciones de Estilo

**Estilo:** Vintage children's storybook illustration, 1950s book art style
**Técnica:** Watercolor and colored pencil texture
**Colores:** Warm muted colors
**Fondo:** Soft cream/beige background
**Vista:** Portrait view, upper body
**Formato:** PNG con fondo transparente o de color crema
**Tamaño:** 512x512px o similar (square)

## 📝 Prompts para Generar Imágenes

Los prompts detallados para cada personaje están en:
`Backmielda_new/data/fiveFromEarthCharacters.js`

Cada personaje tiene un campo `imagePrompt` con el prompt completo para Fal.ai o DALL-E.

## 🔄 Cómo Usar

1. Las imágenes se cargan automáticamente desde esta carpeta
2. Los nombres de archivo deben coincidir exactamente con los especificados
3. Si una imagen no existe, el sistema funcionará sin ella (sin errores)
